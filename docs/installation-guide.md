# Installation Guide
## PKT PLTB Archimedes Spiral Wind Turbine

## Safety Notice

⚠️ **WARNING**: Installation of wind turbines involves working at heights and with electrical systems. Always follow local building codes and electrical regulations. Professional installation is strongly recommended.

### Required Qualifications
- Licensed electrician for electrical connections
- Structural engineer approval for foundation design
- Building permit from local authorities
- Grid connection approval (if applicable)

---

## Pre-Installation Planning

### 1. Site Assessment

#### Wind Resource Evaluation
- Measure average wind speeds for at least 3 months
- Identify prevailing wind directions
- Minimum average wind speed: 4 m/s recommended
- Use wind mapping tools or local meteorological data

#### Site Selection Criteria
✓ Clear airflow from multiple directions
✓ Minimum obstacles within 30m radius
✓ Access for installation equipment
✓ Proximity to electrical connection point
✓ Compliance with local zoning regulations
✓ Minimal noise impact to neighbors

#### Clearance Requirements
- Horizontal clearance: 3x rotor diameter from obstacles
- Vertical clearance: Rotor height + 2m above nearest obstacle
- Distance from property boundaries: Minimum 5m
- Distance from power lines: As per local regulations

### 2. Regulatory Requirements

#### Permits and Approvals
1. Building permit
2. Electrical installation permit
3. Grid connection agreement (if applicable)
4. Environmental assessment (if required)
5. Aviation authority clearance (if near airports)
6. Neighbor notification (as per local requirements)

#### Documentation Needed
- Site plan
- Foundation design
- Electrical schematic
- Structural calculations
- Noise impact assessment
- Visual impact assessment (if required)

### 3. Tools and Equipment Required

#### Installation Tools
- Concrete mixer or ready-mix concrete
- Post hole digger or auger
- Level (laser level recommended)
- Torque wrench (calibrated)
- Socket wrench set
- Drill with masonry bits
- Wire strippers and crimpers
- Multimeter
- Safety harness and fall protection
- Ladder or lift equipment
- Crane or hoist (for larger models)

#### Consumables
- Anchor bolts (provided with turbine)
- Concrete (type and quantity specified in manual)
- Electrical conduit
- Electrical wire (sized per specifications)
- Grounding rod and wire
- Sealant and weatherproofing materials
- Cable ties and mounting hardware

## Installation Procedure

### Step 1: Foundation Installation

#### 1.1 Mark Foundation Location
- Use site plan to locate foundation
- Mark anchor bolt positions precisely
- Verify clearances and orientations
- Check for underground utilities (call before digging)

#### 1.2 Excavate Foundation
**Small Model (PKT-AS-2026-S)**
- Diameter: 60 cm
- Depth: 50 cm
- Volume: ~0.14 m³ concrete

**Medium Model (PKT-AS-2026-M)**
- Diameter: 80 cm
- Depth: 80 cm
- Volume: ~0.40 m³ concrete

**Large Model (PKT-AS-2026-L)**
- Diameter: 120 cm
- Depth: 120 cm
- Volume: ~1.36 m³ concrete

#### 1.3 Install Anchor Bolts
1. Create anchor bolt template (provided)
2. Position template at exact location
3. Insert anchor bolts through template
4. Ensure bolts are perfectly vertical
5. Support template securely

#### 1.4 Pour Concrete
1. Use concrete mix: Minimum 25 MPa (3,600 psi)
2. Pour concrete in layers
3. Vibrate to remove air pockets
4. Check anchor bolt alignment continuously
5. Finish surface level and smooth
6. Cure concrete for minimum 7 days
7. Keep concrete moist during curing

### Step 2: Electrical Preparation

#### 2.1 Install Electrical Conduit
- Run conduit from foundation to connection point
- Use weatherproof conduit rated for outdoor use
- Minimum burial depth: 45 cm (18 inches)
- Seal all conduit entries
- Label all conductors

#### 2.2 Install Grounding System
1. Drive grounding rod at least 2.4m (8 ft) into ground
2. Connect grounding rod to turbine frame
3. Use minimum 6 AWG copper wire
4. Test ground resistance (should be < 25 Ω)
5. Document grounding installation

#### 2.3 Prepare Electrical Connections
- Size wires according to turbine specifications
- Install circuit breakers per local code
- Prepare inverter location (if applicable)
- Install disconnect switch
- Verify all electrical components are rated correctly

### Step 3: Tower Assembly

#### 3.1 Unpack Components
1. Inspect all components for damage
2. Verify all parts against packing list
3. Check hardware and fasteners
4. Review assembly manual
5. Organize parts for assembly

#### 3.2 Assemble Tower Sections (if applicable)
1. Lay tower sections on flat surface
2. Align bolt holes
3. Insert bolts but do not fully tighten
4. Check alignment and squareness
5. Torque bolts to specification (sequential pattern)
6. Verify tower is straight

#### 3.3 Install Tower on Foundation
**For Small Models:**
1. Position tower over anchor bolts
2. Level tower in both directions
3. Install washers and nuts
4. Hand-tighten all nuts
5. Verify level again
6. Torque nuts in star pattern
7. Apply thread locking compound

**For Medium and Large Models:**
1. May require crane or lifting equipment
2. Attach lifting equipment to tower
3. Position tower over anchor bolts
4. Guide tower into position
5. Install base plate and hardware
6. Level and secure as above

### Step 4: Turbine Installation

#### 4.1 Install Generator Assembly
1. Remove shipping restraints from generator
2. Check for shipping damage
3. Verify rotation is free (manual turn test)
4. Install generator onto tower mounting
5. Secure with provided bolts
6. Torque to specifications
7. Connect electrical cables

#### 4.2 Install Spiral Blade Assembly
**Critical: Blade installation requires calm weather (wind < 5 m/s)**

1. Verify blade is undamaged
2. Align blade with mounting hub
3. Install mounting bolts
4. Torque bolts in star pattern to specification
5. Verify blade rotates freely
6. Check balance (blade should stop in random positions)
7. Install safety locking mechanism

#### 4.3 Install Control System
1. Mount controller box to tower or nearby structure
2. Weatherproof mounting location
3. Connect generator cables to controller
4. Connect communication cables
5. Install emergency stop switch (accessible location)
6. Verify all connections are secure

### Step 5: Electrical Connections

⚠️ **Licensed Electrician Required for This Step**

#### 5.1 DC Connections (if applicable)
1. Connect generator to controller
2. Verify polarity (+ and -)
3. Use proper crimping tools
4. Apply dielectric grease to connections
5. Test voltage output with multimeter

#### 5.2 AC Connections (Grid-Tie or Inverter)
1. Connect inverter to controller
2. Connect inverter to AC distribution panel
3. Install required circuit protection
4. Verify voltage and frequency settings
5. Configure inverter parameters
6. Test with grid disconnected

#### 5.3 Grounding and Bonding
1. Connect all metal components to ground
2. Bond control enclosure to ground
3. Bond tower to ground system
4. Verify continuity of all ground connections
5. Test ground resistance

### Step 6: System Configuration

#### 6.1 Controller Setup
1. Power on control system
2. Configure wind turbine model parameters
3. Set cut-in and cut-out speeds
4. Configure brake system
5. Set power output limits
6. Enable safety features

#### 6.2 Communication Setup
1. Connect to local WiFi or cellular network
2. Configure IoT connection
3. Install mobile app
4. Register turbine serial number
5. Test remote monitoring
6. Set up alert notifications

#### 6.3 Calibration
1. Verify wind speed sensor (if installed)
2. Calibrate power output readings
3. Test brake system operation
4. Verify emergency stop function
5. Document all settings

### Step 7: Testing and Commissioning

#### 7.1 Pre-Operation Checks
□ All bolts torqued to specification
□ Blade rotates freely
□ No unusual vibrations
□ Electrical connections secure
□ Grounding system tested
□ Safety systems functional
□ Emergency stop accessible
□ Control system responding

#### 7.2 Initial Start-Up
1. Ensure area is clear of people and obstacles
2. Verify weather conditions suitable (wind < 10 m/s)
3. Release blade brake
4. Allow turbine to start automatically at cut-in speed
5. Monitor operation for first 30 minutes
6. Check for vibrations or unusual sounds
7. Verify power output matches wind speed

#### 7.3 Performance Testing
1. Record baseline performance data
2. Test operation at various wind speeds
3. Verify automatic brake at cut-out speed
4. Test emergency stop function
5. Measure noise levels at property boundaries
6. Document all test results

#### 7.4 Grid Connection (if applicable)
1. Notify utility company
2. Wait for final approval
3. Utility inspector verification
4. Enable grid connection
5. Monitor synchronization
6. Verify bidirectional meter operation

## Post-Installation

### 1. Documentation
Create installation file containing:
- Installation photos
- Test results
- As-built drawings
- Electrical schematics
- Equipment serial numbers
- Warranty registration
- Maintenance schedule

### 2. Owner Training
Train owner/operator on:
- Normal operation
- Monitoring system
- Emergency procedures
- Basic maintenance
- Troubleshooting
- Who to call for service

### 3. First Week Monitoring
- Inspect daily for first week
- Check all bolts and connections
- Verify no unusual vibrations
- Monitor power output
- Listen for unusual sounds
- Document any issues

### 4. 30-Day Follow-Up
- Comprehensive system inspection
- Re-torque all bolts
- Check electrical connections
- Verify calibration
- Review performance data
- Address any concerns

## Troubleshooting Guide

### Turbine Not Starting
- Check wind speed (must exceed cut-in speed)
- Verify brake is released
- Check emergency stop switch
- Verify power to controller
- Check for error codes

### Low Power Output
- Verify wind speed is adequate
- Check for obstructions to airflow
- Inspect blades for damage
- Verify electrical connections
- Check inverter settings

### Excessive Noise
- Check blade balance
- Inspect mounting bolts
- Check for loose components
- Verify blade damage
- Contact service technician

### System Errors
- Check error code in manual
- Verify all sensors functioning
- Reset controller
- Check communication connections
- Contact technical support
